Percentage Increase

Calculate the percentage increase from one value to another.

Calculate Percentage Increase

?%

Formula: ((New - Original) / |Original|) × 100

Example

If a product's price rises from $80 to $100, the percentage increase is:

((100 - 80) / 80) × 100 = 25%

Percentage Decrease

Calculate the percentage decrease from one value to another.

Calculate Percentage Decrease

?%

Formula: ((Original - New) / |Original|) × 100

Example

If a product's price falls from $100 to $75, the percentage decrease is:

((100 - 75) / 100) × 100 = 25%

Percentage Difference

Calculate the percentage difference between two values.

Calculate Percentage Difference

?%

Formula: (|Value1 - Value2| / ((Value1 + Value2) / 2)) × 100

Example

The percentage difference between 60 and 80 is:

(|60 - 80| / ((60 + 80) / 2)) × 100 = 28.57%

How to Use the Percentage Calculator

This calculator helps you solve various percentage-related problems quickly and easily.

Basic Percentage Calculations

  • To find X% of Y: Enter the percentage in X and the value in Y, then click Calculate.
  • To find what percentage X is of Y: Enter the values in their respective fields and click Calculate.

Percentage Increase/Decrease

  • Percentage Increase: Enter the original value and the new (higher) value.
  • Percentage Decrease: Enter the original value and the new (lower) value.

Percentage Difference

Use this when comparing two values without considering which came first. It's useful for comparing measurements, statistics, etc.

Did You Know?

The word "percent" comes from the Latin phrase "per centum," which means "by the hundred." This is why we write 100% when referring to a whole.
